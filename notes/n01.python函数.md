### `python`函数与列表解析式表达式

如下 `intersect()`函数的可以使用列表解析表达式来替代：

```python
def intersect(s1, s2):
    res = []
    for x in s1:
        if x in s2:
            res.append(x)
    return res


if "__main__" == __name__:
    s1 = "scam"
    s2 = "pcsm"

    print(intersect(s1, s2))          // ['s', 'c', 'm']
    print([x for x in s1 if x in s2]) // todo: ['s', 'c', 'm']
    pass
```
